「大湖鄉草莓直送」是一個致力於推廣苗栗大湖無農藥草莓的單頁式農產品電商與形象網站。本專案為期末作業 (Final Homework)，網站主打「給孩子最天然的健康點心」，提供產地透明、全程冷鏈直送的優質草莓訂購服務。
專案特色現代化單頁設計 (Landing Page)：流暢的滾動與導覽體驗，使用者可快速瀏覽商品、品牌故事與常見問題。  痛點與解決方案展示：清晰列出消費者購買草莓時的常見困擾（如農藥殘留、品質不穩），並對應提供專業保證。  互動式詢價表單：內建完整的購物與詢價表單，支援商品下拉選單與特殊需求備註。  RWD 響應式設計：完美適應各種裝置螢幕尺寸（手機、平板、桌機），確保最佳瀏覽體驗。  無農藥與品質保證：強調 100% 無農藥認證、產地透明（QR Code 追溯）以及 30 天滿意保證。  
技術堆疊核心技術：100% 使用純 HTML、CSS、Vanilla JavaScript 構建，無依賴大型前端框架。  部署平台：透過 GitHub Pages 進行靜態網頁部署。  
 網站架構  首頁視覺 (Hero Section)：明確的核心標語，傳遞「新鮮、無農藥、直送」的價值主張。  痛點與承諾 (Problems & Solutions)：建立與消費者的共鳴與信任。  商品展示 (Products)：展示包含鮮採新鮮草莓禮盒等商品。  競品比較 (Comparison)：以清晰的表格呈現與傳統超市及一般農產品網站的各項差異。  品牌故事 (Story)：講述苗栗大湖鄉在地農民的用心經營與自然生態管理理念。  常見問題 (FAQ)：互動式的折疊面板，解答關於農藥、配送、保存及食用上的常見疑惑。  聯絡與訂單表單 (Contact Form)：供消費者快速下單、詢價及訂閱新品通知的互動介面。
